Privacy Policy

Vigil Network Monitor — Privacy Policy

Effective date: 2026 · Version 2.0

What Vigil Is

Vigil is a locally-hosted network monitoring application. It runs entirely on your own computer or server. There is no hosted service, no cloud component, and no connection to any third-party server operated by this project.

Data We Collect

Vigil collects no personal data. The only information it stores is the network configuration you enter yourself — room names, device names, and IP addresses — along with your hashed access PIN. This information is stored in a file called config.json on your machine and is never transmitted anywhere.

Data We Do Not Collect

Vigil does not collect, transmit, or store: usage analytics, telemetry, crash reports, device fingerprints, user identities, browser information, or any form of tracking data. There are no cookies beyond the session token used to keep you logged in, and that token lives only in your browser's memory — it is not persisted to disk.

Network Activity

The only outbound network activity Vigil performs is the ICMP ping, UDP probe, and SSH connection requests you configure — directed at the devices you specify, on your own network. Vigil never connects to any address you have not explicitly configured.

The application loads fonts from Google Fonts CDN on first load. If you prefer fully offline operation, you can disable this by editing index.html to remove the Google Fonts link.

Security

All communication between your browser and Vigil is encrypted via HTTPS using a locally-generated self-signed certificate. Your PIN is stored as a PBKDF2-SHA256 hash — never in plaintext. Sessions expire after 8 hours and are not stored on disk.

Third Parties

Vigil has no relationship with any third-party data processor. No data from Vigil is shared with, sold to, or accessible by any third party.

Your Rights

Since Vigil stores no personal data and operates entirely on your own infrastructure, all data is inherently under your control. You can delete config.json at any time to remove all stored configuration.
